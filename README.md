# Guia Básico de Markdown

Markdown é uma linguagem de marcação simples usada para formatar textos no **GitHub**, documentação, README de projetos, wikis e muito mais.

---

# 1. Títulos (Headings)

Use `#` para criar títulos.

```
# Título 1
## Título 2
### Título 3
#### Título 4
```

Resultado:

# Título 1

## Título 2

### Título 3

#### Título 4

---

# 2. Texto em Negrito e Itálico

```
**Negrito**
*Itálico*
***Negrito e Itálico***
```

Resultado:

**Negrito**

*Itálico*

***Negrito e Itálico***

---

# 3. Listas

## Lista não ordenada

```
- Item 1
- Item 2
- Item 3
```

Resultado:

* Item 1
* Item 2
* Item 3

## Lista ordenada

```
1. Primeiro
2. Segundo
3. Terceiro
```

Resultado:

1. Primeiro
2. Segundo
3. Terceiro

---

# 4. Links

```
[Google](https://google.com)
```

Resultado:

[Google](https://google.com)

---

# 5. Imagens

```
![Texto da imagem](https://via.placeholder.com/150)
```

---

# 6. Código

## Código em linha

```
Use o comando `ls -la`
```

Resultado:

Use o comando `ls -la`

## Bloco de código

````markdown
```bash
ls -la
cd /home
```
````

Resultado:

```bash
ls -la
cd /home
```


# 9. Checklist

Muito usado em issues do GitHub.

```
- [x] Tarefa concluída
- [ ] Tarefa pendente
```

Resultado:

* [x] Tarefa concluída
* [ ] Tarefa pendente
